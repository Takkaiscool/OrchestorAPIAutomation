# OrchestorAPIAutomation

Swagger-driven API automation blueprint for:

- Functional test orchestration
- Schema diff and impact analysis
- Dependency-aware execution planning
- Payload fuzzing and DB validations
- Playwright-based execution/reporting
- JMeter performance plan generation from Swagger

## New capability

Provide a Swagger URL and Copilot can now also generate a JMeter `.jmx` performance test plan that covers all discovered endpoints with parameter and auth placeholders.
