# OrchestorAPIAutomation

This repository now provides Codex skills for migrating legacy Java-based test automation into Playwright TypeScript.

## Included skills

- `skills/public/java-selenium-to-playwright-ts`: migrate Java Selenium UI automation to Playwright TypeScript.
- `skills/public/rest-assured-to-playwright-api-ts`: migrate Rest Assured API automation to Playwright TypeScript.
- `skills/public/db-to-playwright-ts`: migrate JDBC and SQL-backed validation scripts to Playwright TypeScript support code.
